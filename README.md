## Multi-Stage Lead Scoring & Automated Routing System


An end-to-end workflow that qualifies incoming leads, scores them based on business rules and routes high-value prospects instantly.

###  What It Does

- Receives leads through webhook/forms
- Cleans and standardizes lead data
- Scores leads using custom logic
- Separates Hot vs Cold leads
- Sends instant alerts for priority leads
- Logs all leads in Google Sheets

###  Tech Stack

- n8n
- JavaScript (Code Nodes)
- Google Sheets
- Gmail
- Slack
- Webhooks

###  Business Value

- Reduces manual lead sorting
- Faster response to high-value prospects
- Prevents missed opportunities
- Creates repeatable sales workflow
